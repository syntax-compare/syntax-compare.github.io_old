3.2. for-loop (with index)
val fruits = listOf("apple", "banana", "mango")

// option 1
for (i in fruits.indices) {
    println("$i: ${fruits[i]}")
}

// option 2
for ((i, fruit) in fruits.withIndex()) {
    println("$i: $fruit")
}

// option 3
fruits.forEachIndexed { i, fruit ->
    println("$i: $fruit")
}

// option 4 (for specific range)
for (i in 0..2) {  // [0, 2]
    println("$i: ${fruits[i]}")
}

0: apple
1: banana
2: mango
